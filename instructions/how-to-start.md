# Как начать

1. Клонировать репозиторий
`https://github.com/NataliItaly/Smart-Quiz.git`

2. Перейти в папку
`cd Smart-Quiz`

## Для написания журнала
*оставаться на ветке main*
*main будет содержать только финальную версию проекта.*
*До конца разработки master (main) пустой и содержит только docs: README, журналы, инструкции.*

**Не запускать npm install в main - там пока нет package.json**

## Для разработки

**Перейти на ветку develop**

`git checkout develop`

1. Установить зависимости
`npm install`
- установит зависимости
- запустит prepare
- Husky создаст .husky/ hooks

**Убедиться, что Node версия одинаковая**
Node 18+ / 20+


2. Создать feature-ветку
`git checkout -b feature`

3. Запустить проект
`npm run dev`

#### Нужно ли запускать lint вручную?

Теоретически — нет, Husky настроен на pre-commit с:  `"check": "eslint . && tsc --noEmit"`, при каждом git commit ESLint проверится автоматически, если есть ошибки — commit не пройдет

На всякий случай перед созданием Pull Request выполнить:
`npm run check` (та же команда запускается husky перед каждым комитом)
Если нет ошибок — создаём PR.


### Убедиться, что Husky работает

Можно проверить:
`ls .husky`
И попробовать сделать commit с ошибкой — если commit блокируется, всё работает.
