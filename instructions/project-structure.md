# Smart Quiz (JS / HTML / CSS)
Классический quiz-тренажёр, но с разными уровнями сложности, таймером, прогрессом, историей попыток,
режимом “Explain my mistake” через AI


Вопросы - в JSON
Результаты — в localStorage
Использовать AI только для объяснения ошибки, упрощения объяснения, подсказки.
Пользователь отвечает → если ошибка → кнопка "Explain" → текстовое объяснение.

## Предлагаемая структура страниц
1. Login Page
- Псевдо-логин
- Username сохраняется в localStorage
- Нет настоящей регистрации
- Нет паролей
- Нет backend

2. Start Screen (Dashboard)
Здесь будут:
- Выбор темы
- Выбор сложности
- Random mode
- Кнопка Start
- Отображение текущего username
- Кнопку "Statistics"
- Кнопку "Logout"

3. Quiz Screen
- Показываем вопрос
- Выбор варианта
- Кнопка Check
- После проверки:
- Если верно → Next
- Если неверно → Try again + Explain, Кнопка Explain доступна только после проверки
- Показывать правильный ответ визуально
- Disable варианты после проверки

4. User Statistics
- Что можно хранить:
- Общее количество вопросов
- Процент правильных
- Статистика по темам
- Статистика по сложности
- История последних сессий
- Всё можно хранить в localStorage.

5. Routing
Нужно продумать навигацию:
- /login
- /dashboard
- /quiz
- /statistics

### State Management
Нужно продумать:
- Где хранится текущий пользователь?
- Где хранится текущий quiz session?
- Как передаются выбранные фильтры?
- Без фреймворков это нужно продумать заранее.
